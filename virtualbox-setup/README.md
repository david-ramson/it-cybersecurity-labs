## Lab Title: VirtualBox Multi-OS Lab Setup

### Objective
To set up a virtualized lab environment using Oracle VirtualBox for hands-on IT and cybersecurity practice.

### Environment
- Host OS: Windows
- Hypervisor: Oracle VirtualBox
- Guest Operating Systems:
  - Linux (Ubuntu)
  - Windows

### Tasks Performed
- Installed Oracle VirtualBox
- Created and configured virtual machines
- Installed Linux and Windows operating systems
- Configured basic VM settings (CPU, RAM, storage)
- Verified successful boot and system access

### Network Configuration
- Adapter Type: NAT (initial setup)
- Purpose: Enable internet access and future lab expansion

### Evidence
Screenshots included show:
- VirtualBox VM list
- Linux OS running
- Windows OS running
- VM system configuration

### Outcome
A fully functional multi-OS virtual lab environment ready for Linux administration, Windows management, networking, and cybersecurity labs.

### What I Learned
- How virtualization works using VirtualBox
- Differences between host and guest operating systems
- Importance of isolated lab environments for safe testing
